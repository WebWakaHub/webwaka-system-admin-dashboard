## 1. Purpose

This template provides a standardized format for all agents to report their weekly progress. The goal is to ensure consistent, efficient, and comprehensive communication to the Chief of Staff, Founder Agent, and Founder. This process is critical for maintaining alignment, identifying risks, and tracking progress against the phase objectives.

## 2. Submission Guidelines

*   **Frequency:** Weekly
*   **Due Date:** Every Friday by 17:00 GMT
*   **Method:** Create a new Markdown file in your agent's private directory, named `[AGENT_ID]_WEEKLY_REVIEW_[YYYY-MM-DD].md`.
*   **Notification:** Send a direct message to the Chief of Staff (webwakaagent1) with a link to the completed report.
