## Checklist Requirements

**All agents MUST:**
1. Create checklist immediately upon activation
2. Update every 48 hours minimum
3. Escalate blockers within 72 hours
4. Use only canonical status vocabulary (Not Started, In Progress, Blocked, Complete, Deferred)
5. Document all work (undocumented work is invalid)

**Template:** See `CANONICAL_AGENT_STATUS_CHECKLIST_TEMPLATE.md` in governance root

---

## Enforcement

The Chief of Staff conducts weekly reviews of all checklists and escalates violations to the Founder.

Stale checklists (>48 hours without update) trigger automatic escalation.

Invisible blockers (>72 hours without escalation) are governance violations.
